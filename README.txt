Этот фреймворк открывает Google cloud calculator и проверяет, соответсвует ли цена в калькуляторе цене в присланном письме, а также цене, полученной при ручном заполнении.
Использованы:
WebDriverIO для управления коннекторам к браузерам
PageObject/ Page Factory для абстракций страниц
Business model для бизнес-объектов необходимых сущностей
Файлы с тестовыми данными для разных окружений
Suites для smoke тестов и всех типов тестов 
Добавлен spec-reporter
При падении теста делается скриншот с датой и временем 
Фреймворк имеет возможность запуска с Jenkins и параметризацией браузера, тест suites. Результаты тестов есть на графике джобы, скриншоты заархивированы как артефакты
